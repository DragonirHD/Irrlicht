Website Info

This website is being made with Angular and Angular Material.

The components are all mostly made in house and without any code written by AI. This is done to build up and increase our own knowledge about web design and to give the website a unique and custom feel.

The website is built with a responsive design that supports mobile devices of sizes down to 360px and also comes with dark-/light-mode that automatically adjusts to the users preferences.
